Le module runExe permet de lancer et d'arrêter des applications :


SARAH, démarre XBMC

SARAH, arrête l'application Write

SARAH, lance le script d'ouverture

SARAH, met à jour les applications


La liste des applications est parametrable dans un fichier txt proglist.txt modifiable dans le répertoire runExe.

Ce plugin est mis à disposition comme exemple et peut être entièrement personnalisé, en particulier parce que le répertoire bin/ contient plusieurs programmes pour arrêter un processus, ou verrouiller l'ordinateur et autres.

Remarque: runExe utilise un shell pour tuer les taches afin de les fermer. Pour arrêter un processus executé via un script ou un batch, il faut créer l'équivalent pour la fermeture et lancer le script. Le fichier proglist.txt montre un exemple
